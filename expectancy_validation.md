# Expectancy Validation

## Formulas
```
WinRate = WinCount / TotalTrades * 100
AvgWin = GrossProfit / WinCount
AvgLoss = GrossLoss / LossCount
Expectancy = WinRate * AvgWin + (1 - WinRate) * AvgLoss
AvgTrade = (GrossProfit + GrossLoss) / TotalTrades
```
