# Profit Factor Validation

## Formulas
```
GrossProfit = sum(NetPnL_i) for all i where NetPnL_i > 0
GrossLoss = sum(NetPnL_i) for all i where NetPnL_i < 0
AbsLoss = |GrossLoss|  (make it positive)
ProfitFactor = GrossProfit / AbsLoss
```
